# iasChange - Application de change CFA ↔ Ouguiya

## 🌐 Accès au site
[https://iasgithub.github.io/ias-change-2/](https://iasgithub.github.io/ias-change-2/)

## 📝 Description
Application web de change de devises permettant de convertir le Franc CFA en Ouguiya (MRO) et vice-versa.

## ✨ Fonctionnalités
- Conversion en temps réel CFA ↔ MRO
- Tableau des prix pré-définis
- Interface responsive (mobile, tablette, PC)
- Panneau administrateur sécurisé par mot de passe

## 🔧 Installation locale
1. Clonez ce dépôt
2. Ouvrez `index.html` dans votre navigateur

## 🔑 Accès administrateur
- Double-cliquez sur le titre "💱 iasChange"
- Mot de passe par défaut : `admin123`

## 🔒 Sécurité
- Les données sensibles sont stockées dans `config.js` (à configurer localement)
- Le panneau admin est protégé par mot de passe

## 🛠️ Technologies utilisées
- HTML5 / CSS3 / JavaScript Vanilla
- GitHub Pages pour l'hébergement
